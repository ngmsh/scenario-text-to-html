# scenario-text-to-html
テキスト状態で作った脚本データを、脚本っぽいデザインのHTMLに変換する

## 書式

* 作品名はファイル名を採用
* 場の表示は□で区切る
* 台詞は以下のいずれか。書式は混在していても問題ない。
  * 名前: 台詞
  * 名前「台詞」
* ト書きは4文字下げ

```例
□第一場

    どこか不思議な世界不思議な時間、何かが起こりそうな雰囲気。
    倒れていた一郎は意識を取り戻し、周りを見回す
    
一郎: こ、ここはいったいどこなんだ？ 俺の身になにが……

    と、二郎現れる

二郎「ついにこの地にやってきたようだな、一郎」
```
